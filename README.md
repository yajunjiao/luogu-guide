# 前言
最近看到很多新手在讨论区问关于如何使用洛谷的问题。对于老手来说，这样的问题自然是很烦人，于是我开始准备写这样一个指南，来从一个用户的角度介绍洛谷的各种功能。

本生存手册**非官方**版本，我会在合适的位置给出官方文档的链接以供参考。
